Список изменений версий плагина MKJ SimpleCaptcha для MaxSite CMS.

= 0.2.1 =
 * Удалён произвольный префикс - целиком и полностью - как вещь ненужная;
 * CSS подругжаются в <head> на страницах записей;
 * Стандартный формат картинок теперь .png;
 * Созданы набор картинок.

= 0.2b =
 * Добавлен новый тип капчи: без использования JavaScript (старый тип доступен в настройках);
 * Добавлено произвольное количество картинок;
 * Добавлено произвольное расширение картинок;
 * Добавлен произвольный префикс всех ID элементов плагина;
 * Теперь можно изменить текст ошибки;
 * Теперь можно показывать/скрывать сообщение для пользователей без JavaScript в версии с использованием JS, а также редактировать это сообщение;
 * Добавлена поддержка произвольного CSS (на данный момент плохая).

= 0.1.2 =
Исправлена критическая ошибка, при которой капча не проверялась. 

= 0.1.1 =
 * Исправлены некоторые недочёты кода;
 * Добавлено сообщение о неверно выбранной картинке;
 * Теперь, если активировать плагин и не открывать страницу настроек, ошибки не появляются;
 * При наличии уже имеющейся капчи - она удаляется;
 * Для пользователей с отключенным JavaScript добавлено информационное сообщение;
 * Изменено название плагина в связи с тем, что плагин (почти) с таким названием уже есть.

= 0.1 =
Первая версия плагина.
