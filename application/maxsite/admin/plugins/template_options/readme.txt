Данный плагин позволяет вынести настройки шаблона в админ-панель.

Для этого следует создать файл options.php в каталоге текущей темы.

После активации плагина и при условии существования файла options.php в админке 
появится пункт меню «Настройка шаблона». Функциональность шаблона определяется 
индивидуально.
